Introducción a Python

En esta sección se introduce Python y se explica cómo instalarlo y configurarlo en tu ordenador. También se cubren los conceptos básicos de programación, como variables y estructuras de control.

Variables y tipos de datos

En esta sección se cubren los diferentes tipos de datos que puedes manejar en Python, como cadenas de texto, números y booleanos. También se explica cómo utilizar variables para almacenar y manipular datos.

Condicionales y bucles

En esta sección se cubren las estructuras de control de flujo de Python, como los condicionales if-else y los bucles for y while. También se explica cómo utilizar estas estructuras para crear programas más complejos.

Listas y tuplas

En esta sección se cubren las listas y tuplas en Python, que son estructuras de datos que permiten almacenar múltiples elementos en una sola variable. Se explica cómo utilizar estas estructuras y cómo manipular sus elementos.

Diccionarios

En esta sección se cubren los diccionarios en Python, que son estructuras de datos que permiten almacenar pares clave-valor.

Funciones

En esta sección se cubren las funciones en Python, que son bloques de código que pueden ser reutilizados en diferentes partes del programa. Se explica cómo crear y utilizar funciones, así como también se habla de los argumentos y valores de retorno de las mismas.

Programación orientada a objetos

En esta sección se cubre la programación orientada a objetos en Python, que es un paradigma de programación que permite estructurar el código de manera más organizada y modular. Se explica cómo crear clases, objetos y métodos en Python.

Proyectos prácticos

En esta sección se presentan algunos proyectos prácticos que puedes realizar para aplicar lo que has aprendido en el curso. Entre ellos, se incluyen proyectos como un juego de adivinanza y una aplicación de gestión de tareas.

Introducción a C++

En esta sección se presenta una introducción al lenguaje de programación C++, sus características principales y cómo se diferencia de otros lenguajes. También se aborda la configuración del entorno de desarrollo y la forma de compilar y ejecutar programas.

Tipos de datos

En esta sección se cubren los tipos de datos que se utilizan en C++ y su forma de declaración. También se enseña a trabajar con variables y constantes, y a realizar operaciones aritméticas con los distintos tipos de datos.

Estructuras de control de flujo

En esta sección se cubren las estructuras de control de flujo en C++, como los bucles for, while y do-while, y las sentencias de control if, else if y switch. También se explica cómo se puede utilizar la sentencia break y continue dentro de los bucles.

Arrays

En esta sección se cubre el uso de arrays en C++, tanto unidimensionales como multidimensionales. Se explica cómo se declaran, inicializan y acceden a los elementos de los arrays.

Funciones

En esta sección se cubren las funciones en C++, que son bloques de código que pueden ser reutilizados en diferentes partes del programa. Se explica cómo crear y utilizar funciones, así como también se habla de los argumentos y valores de retorno de las mismas.

Punteros

En esta sección se cubren los punteros en C++, que son variables que contienen la dirección de memoria de otra variable. Se explica cómo declarar y utilizar punteros, así como también se habla de la aritmética de punteros y los punteros a funciones.

Clases y objetos

En esta sección se cubre la programación orientada a objetos en C++, que se basa en la creación de clases y objetos. Se explica cómo se definen las clases, cómo se crean objetos a partir de ellas y cómo se pueden utilizar los distintos elementos de una clase, como atributos y métodos.

Proyectos prácticos

En esta sección se presenta una serie de proyectos prácticos que permitirán aplicar los conceptos aprendidos en el curso. Entre ellos se encuentra la creación de un programa de cálculo de promedio de notas, la implementación de una lista de tareas con clases y objetos, y la creación de un juego de adivinanza de números.

Curso de HTML y CSS

En este curso de HTML y CSS aprenderás los conceptos básicos de la creación de páginas web utilizando los lenguajes HTML y CSS. Con este conocimiento podrás crear tus propias páginas web y darles formato y estilo de manera profesional.

Este video te ofrece una introducción al curso de HTML y CSS y te muestra algunos ejemplos básicos de cómo se estructura una página web utilizando HTML y se le da estilo utilizando CSS.

Introducción a HTML y CSS

En esta sección se cubren los conceptos básicos de HTML y CSS, como la estructura básica de un documento HTML, las etiquetas, los atributos, los selectores CSS, las propiedades y los valores. También se explica cómo crear enlaces y cómo utilizar imágenes en una página web.

Formato de texto con CSS

En esta sección se cubren las propiedades CSS que se utilizan para dar formato al texto en una página web. Se explican las propiedades para crear estilos de texto personalizados, como tamaños de fuente, colores, estilos de fuente y más.

Diseño de página con CSS

En esta sección se cubren las técnicas de diseño de página con CSS, como la creación de diseños de caja, la posición de elementos, la creación de diseños fluidos y responsivos y la creación de menús de navegación.

Tipografía con CSS

En esta sección se cubren las propiedades CSS que se utilizan para dar formato y estilo a la tipografía en una página web. Se explican las propiedades para crear estilos de fuente personalizados, como familias de fuente, pesos de fuente, tamaño de línea y más.

Proyectos prácticos

En esta sección se presentan varios proyectos prácticos que permitirán aplicar los conceptos aprendidos en el curso. Entre ellos se encuentra la creación de una página web personal, la implementación de un menú de navegación y la creación de una página de contacto con un formulario de contacto.
